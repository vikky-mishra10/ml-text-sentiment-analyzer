# ML Text Sentiment Analyzer

This project uses a simple Machine Learning model to analyze text sentiment.  
It demonstrates the complete ML workflow: data preprocessing, feature extraction, model training, and evaluation.

## Features
- Text preprocessing
- TF-IDF feature extraction
- Sentiment classification (Positive / Negative)
- Model evaluation using accuracy

## Technologies Used
- Python
- Scikit-learn
- Pandas

## ML Model
- Logistic Regression

## Project Status
Initial ML version implemented.  
Future improvements include better datasets and model tuning.
